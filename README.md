# Universo Koachify

Presentazione interattiva immersiva del metodo **Koachify Suite** — 3 sistemi, 9 pianeti, 27 pilastri.

**Demo live:** [andrea93koach.github.io/universo-koachify](https://andrea93koach.github.io/universo-koachify/)

## Cos'è

Un'esperienza di vendita immersiva progettata per live, video YouTube e presentazioni one-to-one. Naviga l'universo Koachify esplorando i tre sistemi (Prosperità · Libertà · Serenità) e i loro pianeti.

## Come si usa

Apri `universo-koachify.html` nel browser — nessuna dipendenza esterna, nessun server necessario.

**Navigazione da presenter:**
- `Space` / `→` — rivela il bullet successivo (o avanza alla prossima sezione)
- `←` — torna alla sezione precedente
- `Esc` — torna alla schermata precedente

## Stack

Vanilla HTML5 · CSS3 · JavaScript — zero dipendenze esterne, zero CDN.
